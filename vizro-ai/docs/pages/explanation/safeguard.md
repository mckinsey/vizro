# Code execution and data handling in Vizro-AI

Vizro-AI does **not** execute code generated by a large language model (LLM). Earlier versions asked the model to write a Python charting function and ran it with `exec()`; that path — and its best-effort code safeguard — has been removed.

## How chart creation works now

The model fills a declarative **chart specification** (a [`BaseChartPlan`][vizro_ai.agents.response_models.BaseChartPlan]): a chart type, a mapping of columns to `plotly.express` arguments, and styling options. It never writes or returns Python. Vizro-AI renders that specification with its own trusted `plotly.express` code:

- `result.output.figure(df)` returns a Plotly `go.Figure`.
- `result.output.to_figure_json(df)` returns the figure as JSON — inert data that a frontend can render at runtime, with no code execution anywhere.
- `result.output.code` / `code_vizro` return the equivalent `plotly.express` code as a *string*, templated from the specification for you to read or reuse. Vizro-AI never runs it.

Because the model's output is data rather than code, there is nothing to `exec()` and no denylist to maintain. A malicious specification can at worst describe an odd-looking chart, not run code on your machine. This makes Vizro-AI suitable behind an LLM gateway that disables code execution, and in server or multi-tenant deployments.

## You provide the data in the right shape

Vizro-AI does not transform data. Any aggregation, filtering, sorting or reshaping must be done **before** you pass the dataframe. For example in SQL, or in a separate data preparation step. Vizro-AI maps the columns of the dataframe you give it onto a chart, and expects the data to already be in the shape you want to plot.

## Still your responsibility

- **Review the model's output.** The chart specification (and the `code` string, if you use it) is generated by an LLM; review it before relying on it in sensitive contexts.
- **Trust your data source.** Vizro-AI reads the dataframe you provide, so supply it from a source you trust.
